Variables

Var

Var a día de hoy es una mala practica ya que asignar variables con var genera errores más que todo porque estas son variables que no respetan un bloque si no que se asignan de manera global.

Let

Es la forma correcta de declarar variables ya que estas se guardan en el ámbito de bloque lo que no genera errores. 

          
            let nombre = "Ale";

Constantes

Las constantes presentan un ámbito de bloque (block scope) la particularidad es que el valor que se le asigne no se puede cambiar a través de la reasignación. Las constantes no se pueden redeclarar 

          
           const PI = 3.1416;
           console.log(PI);

Objetos

Un objeto es una colección de propiedades, y una propiedad es una asociación entre un nombre (o clave) y un valor. 

          
            const objeto = {
              nombre: "Ale",
              edad: 35
           }

Arreglos

Un Array es una colección o agrupación de elementos en una misma variable, cada uno de ellos ubicado por la posición que ocupa el array. 

            
            const colores = ["Blanco", "Negro", "Azul"];

             console.log(objeto);
             console.log(colores);

            objeto.correo = "alejozxxz18@gmail.com"; // Para agregar en objetos se utiliza un .conelvalorquequierasagregar
            colores.push = "Naranja"; // Para agregar en arreglos se utiliza .push

            console.log(objeto);
            console.log(colores);
